### dependencies
1. Lombok
2. Spring Web
3. spring data mongodb
4. testcontainers
5. Springdoc - for adding openapi with swagger.

### Commands
```
brew install maven
mvn clean install
mvn clean compile
```
- Start docker
```
docker compose up -d
```

- Swagger Docs. http://localhost:8080/api-docs
- API Docs. http://localhost:8080/swagger-ui.html
